# Threat Matrix

Real-time intelligence interface for tracking threat logs, CVEs, and infrastructure vulnerabilities. Built for security researchers who need a clean, fast tool without the bloat.

## What's inside
*   **Custom Feeds:** Track and filter incoming threats in real-time based on your parameters.
*   **Quick CVE Lookups:** One tap to pull up exploit analysis on Google or GitHub.
*   **Secure Auth:** Token-based access to keep your private tracking parameters locked down.

## How to Install (Sideloading)
I skipped the Google Play Store because I'm not paying a $25 upfront fee just to host a free, open-source project. That means you'll need to grab the APK directly from the source here.

1. Head over to the [Releases](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME/releases) page and download the latest `app-release.apk`.
2. Tap the file to install it.
3. Your phone will probably yell at you about installing from "Unknown Sources." Just tap **Settings > Allow from this source** and you're good to go.

## Security & Privacy
*   **Zero Telemetry:** I don't track your device, your usage, or your logs. Period.
*   **Obfuscated Builds:** The release APK is minified and obfuscated using R8. It keeps the app size tiny and makes reverse-engineering a pain.
*   **Open Source:** Don't trust me? Good. The entire codebase is right here for you to audit.
